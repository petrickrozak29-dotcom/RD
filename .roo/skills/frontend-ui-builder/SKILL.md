# Frontend Ui Builder

## Instructions

Add your skill instructions here.
When this skill is active:

1. Always design UI using reusable components.

2. Use clean and consistent layout structure.

3. Prioritize responsive design (mobile-first).

4. Keep styling simple and modern.

5. Use clear naming for components and props.

6. Avoid unnecessary complexity in UI logic.

7. Ensure good user experience:
   - Easy navigation
   - Clear hierarchy
   - Minimal clutter

8. When using React/Next.js:
   - Separate components properly
   - Avoid large monolithic files

9. If improving UI:
   - Focus on clarity and usability
   - Do not redesign everything unnecessarily
