# Ai Feature Builder

## Instructions

When this skill is active:

1. Clearly define:
   - User input
   - AI processing
   - Output format

2. Keep prompts efficient and specific.

3. Avoid unnecessary API calls.

4. Structure AI responses clearly:
   - Easy to read
   - Useful to user

5. For chatbot features:
   - Maintain conversation context
   - Handle edge cases

6. Integrate AI with app features:
   - Planner data
   - User activity
   - Notifications

7. Optimize performance:
   - Minimize latency
   - Cache responses if needed

8. If unclear request:
   - Ask clarification instead of guessing
